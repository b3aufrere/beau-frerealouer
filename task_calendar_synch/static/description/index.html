Project Task and Calendar Meeting Synchronization

Task and Meeting Synchronization Odoo App

Create Meeting from Project Task Automatically

This app allows your users to synchronize Project Task with Calendar Meeting. It will automatically create a calendar meeting once a task will be created. If your project user updates the task then the system will synchronize it with Meeting.

Main Features:

  • Allow Project user(s) to Create and Update Calendar Meeting from Project Task Automatically.
  • Project user(s) can see Meeting Information on the Task Form.
  • User(s) can see Task Information on Calendar Meeting Form.
  • Task Creation and Update Task will be synced with Calendar Meeting Automatically.
  • Note: Once Meeting is linked with the Task you can not DELETE that Meeting.
