Odoo Twilio SMS Gateway

Odoo Twilio SMS Gateway helps you integrate & manage Twilio SMS Accounts operations from Odoo. These apps Save your time, Resources, Effort, and Avoid manually manage multiple Twilio SMS Accounts to boost your business SMS Marketing with this connector.

Create & Configure Twilio Account in Odoo

Following steps to you can create and configure Twilio account in Odoo, 

1. Goto Twilio SMS Gateway App > Configurations > Twilio Accounts Menu.
2. Create new account and fill-up all required informations,
	* API Credentials Informations
	2.1 Account SID, Authtoken, and From Mobile Number you can get from Twilio account. 
	2.2 Login to Twilio >> Account Menu >> Manage Account Menu >> General Setting. There is Account SID and Authtoken so copied them and insert into Odoo.
	2.3 For get From Mobile number goto Develop >> Phone Number >> Manage >> Active Numbers. There is Number so copied it and past in Odoo.
	
	* Other/General Information
	2.5 Test Connection Mobile Number: Insert here your any mobile for testing connection with Twilio.
	2.6 After, Click on "Test Connection" button to send test SMS to given mobile number(Which you inserted into 2.5)
	
	** Confirm Twilio Account in Odoo
	2.7 Click on "Test & Confirm Account" button to send test SMS to Test Mobile Number and Confirm account in Odoo.
	2.8 After, Refresh screen to check Account status will changed to Confirmed
	
	** Get Twilio Account Informations
	2.9 After, Confirmed Twilio Account in Odoo you will see the "Get Account Details" button. Click on it to you will get below informations from Twilio.
	> Account Type, Account Status

Create and Configure Twilio SMS Groups in Odoo

Following steps to you can create and configure Twilio SMS Groups in Odoo, 

1. Goto Twilio SMS Gateway App > Configurations > SMS Groups Menu.
2. Create new SMS Group and fill-up all required informations,
> Group Name, Select recipients

Create and Configure Twilio SMS Templates in Odoo

Following steps to you can create and configure Twilio SMS Templates in Odoo, 

1. Goto Twilio SMS Gateway App > Configurations > SMS Template Menu.
2. Create new SMS Template and fill-up all required informations,
> Template Name and Message.

Send SMS to Recipients

Following steps to you can send SMS from Odoo, 

1. Goto Twilio SMS Gateway App > SMS Menu > Send SMS Menu.
2. Create new SMS record and 
	2.1 Select SMS Account for sending message.
	2.3 Send SMS To:
		2.3.1 Contact: You can send SMS to single contact.
		2.3.2 Multiple Contact: You can send SMS to multiple contacts.
		2.3.3 SMS Group: You can send message to group(which have multiple contact).
		2.3.4 Mobile: You can send SMS to direct mobile number if not exist in Odoo.
	2.4 SMS Template: You can select predefine SMS template so don't need to write same message multiple time.
	2.5 Message: You can write your message for send to your contacts.
3. Click on "Send Message" button to send message to your contacts.

Twilio Informations: After sending messages to Contacts, You will get the information about How many messages sent Successfully / Failed, Total Price(Which charged by Twilio) and Other informations.

Log History: Here you will get the full log information about sent messages.

Confirm Sale/Website Order to Auto Send SMS

Following steps to you can auto send SMS while confirm sale order, 

1. Goto Configuration > Twilio Account. Open any Twilio Account which for you want to enable this feature.
	1.1. There is "Confirm Order to Send SMS?" option need to enable.
	1.2 SMS Template: Select SMS Template.
	>> For template create, goto Configuration >> SMS Template.
	>> Then, Create SMS Template and fillup following informations,
		>> Template Name: i.e Sales Order: Confirmation SMS
		>> Applies To: Select Sale Order
		>> Message: Here type message as you like. Here you can access all Sales Order Model's and Other fields which are linked with Sale Order Model.
		Scroll down to you can see the example SMS Template.
		>> Preview Magic Button: Here you can preview your wrote message for existing sale order. 
		Note: This preview features to not send message to customer, It's only for preview.  
2. After, when you confirm sales/website order that time auto send sms to customer.

Validate Delivery/Picking to Auto Send SMS

Following steps to you can auto send SMS while validate delivery, 

1. Goto Configuration > Twilio Account. Open any Twilio Account which for you want to enable this feature.
	1.1. There is "Validate Delivery to Send SMS?" option need to enable.
	1.2 SMS Template: Select SMS Template.
	>> For template create, goto Configuration >> SMS Template.
	>> Then, Create SMS Template and fillup following informations,
		>> Template Name: i.e Delivery: Send by SMS Text Message
		>> Applies To: Select Transfer
		>> Message: Here type message as you like. Here you can access all Delivery Model's and Other fields which are linked with Delivery Model.
		Scroll down to you can see the example SMS Template.
		>> Preview Magic Button: Here you can preview your wrote message for existing delivery orders. 
		Note: This preview features to not send message to customer, It's only for preview.  
2. After, when you validate delivery order that time auto send sms to customer.
